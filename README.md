# EcoVault ♻️

> **Projeto Integrador - FATEC Indaiatuba** > Um sistema focado na conscientização e gestão de reciclagem de materiais, desenvolvido para facilitar o acesso à informação e promover práticas sustentáveis.

---

## 🛠️ Status do Projeto
![Em Desenvolvimento](https://img.shields.io/badge/Status-Em%20Desenvolvimento-yellow)
> O projeto encontra-se em fase de construção (WIP). Funcionalidades e documentação estão sendo atualizadas constantemente.

---

## 🚀 Tecnologias Utilizadas

### **Front-end**
* **React** + **Vite**
* **TypeScript** & **JavaScript**
* Ferramentas: **Visual Studio Code**, **WebStorm**

### **Back-end**
* **Java** + **Spring Boot** (REST API)
* Ferramentas: **IntelliJ IDEA**, **Visual Studio Code**

### **Base de Dados**
* **MongoDB** (NoSQL)
